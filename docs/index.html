2. Optional SMS import (Android)

Qoondeeye offers an optional SMS import feature to help record supported mobile-money transaction alerts.

Permissions

If you enable SMS import, the app requests Android RECEIVE_SMS so it can detect new supported transaction alerts. We do not request access to read your SMS inbox or history, and we do not browse personal conversations.

What messages are processed

Only supported transaction alerts (such as EVC-style messages) are processed. Other SMS is ignored.

How SMS data is used

We may extract limited details such as amount, transaction type, date/time, and sender or counterparty when available. This is used only to create finance records in the app.

Your control

SMS import is optional and can be turned off in settings at any time.
